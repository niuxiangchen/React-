项目描述
1) 包括用户管理 / 商品分类管理 / 商品管理 / 权限管理等功能模块
2) 前端: 使用 React 全家桶 + Antd + Axios + ES6 + Webpack 等技术
3) 后端: 使用 Node + Express + Mongodb 等技术
